# MAANG DSA in Python 🐍

A comprehensive collection of 300+ Data Structures and Algorithms problems commonly asked in MAANG (Meta, Amazon, Apple, Netflix, Google) and Microsoft interviews. 

## Structure

The repository is structured logically by topic and difficulty level.

### Topics Covered
1. Arrays
2. Strings
3. Linked Lists
4. Stacks & Queues
5. Trees
6. Graphs
7. Dynamic Programming
8. Binary Search
9. Two Pointers & Sliding Window
10. Heap & Priority Queue
11. Backtracking
12. Greedy
13. Tries
14. Bit Manipulation

### Problem Format

Each problem file contains:
- **Problem Statement** & Constraints
- **Brute Force Approach**: Initial thought process and implementation, with Time & Space complexities.
- **Optimal Approach**: The most efficient solution expected in an interview, with Time & Space complexities.
- **Test Cases**: `if __name__ == "__main__":` block with assertions to verify the optimal solution.

### Utilities

- `utils/data_structures.py`: Contains implementations for Tree Nodes, Linked List Nodes, Graphs, Union Find, etc. Import these in your solution files for testing.

## How to use

1. Navigate to the topic you want to practice.
2. Read the problem statement in the file.
3. Try solving it yourself before looking at the `brute_force` or `optimal` functions.
4. Run the file directly to execute the tests: `python 001_two_sum.py`.

## Status

Currently contains core foundational problems fully implemented, and templates for you to practice and fill out the remaining problems up to 300.
